# 📚 Biblioteca Digital - Plataforma de Distribuição de Livros

Projeto de Trabalho de Conclusão de Curso (TCC) - Sistema de Livraria Digital

## 📖 O Projeto

O projeto consiste em uma livraria digital inspirada na plataforma Steam, com foco na distribuição e exposição de livros. O site foi desenvolvido com um design minimalista, priorizando a experiência do usuário e a valorização das obras literárias. O sistema também permite que os usuários exibam suas coleções, compartilhem preferências, expressem opiniões e interajam com outros leitores, consolidando-se como uma alternativa moderna, interativa e inclusiva para todos os públicos.

## 🎯 Objetivo da Proposta

A proposta tem como objetivo facilitar o acesso das pessoas ao consumo de livros de forma moderna, intuitiva e acessível. A plataforma busca resolver dificuldades relacionadas ao acesso a obras literárias, quadrinhos e outras publicações, além de oferecer um espaço para que autores independentes possam expor seus trabalhos, promovendo o conhecimento de novas obras e incentivando a leitura.


## 🚀 Como Começar

1. Clone ou baixe este repositório
2. Abra `inicial/i.html` no navegador para acessar a página inicial
3. Navegue pelas diferentes seções da plataforma

## ✨ Funcionalidades Principais

- 🔐 **Sistema de Login e Autenticação** - Acesso seguro à plataforma
- 📖 **Catálogo de Livros** - Navegação intuitiva por obras literárias
- 👁️ **Visualização de Detalhes** - Informações detalhadas sobre cada livro
- 📕 **Modo de Leitura** - Interface dedicada para leitura
- 🛒 **Seção de Loja** - Compra e distribuição de obras
- 💬 **Interação Social** - Compartilhamento de preferências e opiniões
- 🎓 **Suporte a Autores Independentes** - Plataforma para exposição de trabalhos

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js e Express.js
- **Banco de Dados:** Integração com APIs
- **Design:** UX/UI focado em minimalismo e acessibilidade

## 💡 Conclusão

O projeto mostra como a tecnologia pode ajudar na aproximação entre leitores e autores. Ele cumpre seu objetivo ao unir simplicidade, praticidade e modernidade, criando um espaço digital que incentiva a leitura e valoriza a cultura literária.
